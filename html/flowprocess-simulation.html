Alur Kerja Maju (Simulasi Baru)
Langkah 1: Input PenggunaPilih Material, Volume, dan Jarak
Pengguna memulai simulasi dengan memberikan tiga input utama:
  • Pilihan Material: Memilih antara Amonium Nitrat (AN), Propana (LPG), atau Hidrogen (H₂).
  • Volume (m³): Jumlah volume dari material yang dipilih.
  • Jarak (m): Jarak dari titik pusat ledakan ke lokasi yang ditinjau.
Langkah 2: Properti MaterialNilai default dimuat & dapat diubah
Setelah memilih material, properti kimianya dari literatur secara otomatis dimuat. Pengguna dapat meninjau atau mengubah nilai-nilai ini:
  • Density (ρ): ... kg/m³ (Angka default dan editable angka)
  • ΔHexplosion: ... kJ/kg (Angka default dan editable angka)
  • Ambient Pressure (Pa): 101.325 kPa (Default 101.325)
  • ηE Explosion Efficiency: ... (Default)
  • ETNT: ... kJ/kg (Angka default dan editable angka)
Langkah 3: Kalkulasi IntiHitung semua parameter utama ledakan
Aplikasi menghitung semua parameter kunci ledakan.(W) Mass Material = ρ × Volume(E) Total Energy = W × ΔH_explosion × η_E(W_tnt) Ekuivalen TNT = E / E_TNT(Ze) Scaled Distance = Jarak / (W_tnt)^(1/3)(Ps) Scaled Overpressure = Po / Pa(Po) Crowl = Pa * (1616 * (1+(Ze/4.5)²)) / (√(1+(Ze/0.048)²) * √(1+(Ze/0.32)²) * √(1+(Ze/1.35)²))(Po) Alonso (kPa): Jika Ze < 10 → (1.13e6 * Ze⁻²·⁰¹) / 1000 Jika Ze ≥ 10 → (1.83e5 * Ze⁻¹·¹⁶) / 1000(Po) Sadovski (kPa) = (0.085*r + 0.3*r² + 0.8*r³) * 1000 di mana r = (W_tnt)^(1/3) / Jarak
Langkah 4: Validasi ModelPeriksa rentang valid Ze
Aplikasi memvalidasi nilai `Ze` yang dihitung. Peringatan akan diberikan jika nilai berada di luar rentang yang disarankan untuk setiap model.
  • Crowl: 0.01 ≤ Ze ≤ 100
  • Alonso: 1 ≤ Ze ≤ 200
  • Sadovski: 1 ≤ Ze ≤ 15
Langkah 5: Hasil & AnalisisEstimasi Kerusakan/Cedera dan Grafik
Nilai `Po` yang dihitung digunakan untuk:
  • Estimasi Kerusakan Struktur & Peralatan.
  • Estimasi Konsekuensi Cedera.
  • Plot titik (Ze, Ps) pada Grafik Ps vs. Ze.
Langkah 6: Pencatatan SimulasiPengguna menyimpan hasil ke Log
Pengguna dapat menyimpan set lengkap input dan output sebagai sebuah "Node" di tabel log untuk perbandingan.
Langkah 7: Visualisasi AgregatPerbarui Grafik & Peta Kontur
Data dari semua "Node" di log digunakan untuk secara otomatis menghasilkan:
  • Grafik Tekanan Berlebih vs. Jarak.
  • Peta Kontur Ledakan.
Alur Kerja Terbalik (Dari Data Tersimpan)
Hasil Akhir: Input TerisiFormulir diisi otomatis
Sebagai langkah terakhir dari alur terbalik, bidang input awal (Material, Volume, Jarak) dan Properti diisi dengan data dari Node yang dipilih, memungkinkan pengguna untuk meninjau atau memodifikasi simulasi yang ada.
Langkah 4: Analisis DetailPanel estimasi diperbarui
Aplikasi langsung memperbarui panel "Estimasi Kerusakan" dan "Estimasi Cedera" dengan nilai-nilai dari Node yang dipilih. Titik yang sesuai pada Grafik Ps vs. Ze juga disorot.
Langkah 3: Interaksi LogPengguna memilih 'Node' di tabel
Ketika pengguna mengklik baris "Node" tertentu di dalam tabel log...
Langkah 2: Visualisasi InstanGrafik dan Peta diplot
Berdasarkan data yang diimpor atau dimuat, "Grafik Tekanan Berlebih vs. Jarak" dan "Peta Kontur Ledakan" segera diplot. Data dari Node 1 ditampilkan secara default.
Langkah 1: Memuat DataImpor atau Muat Log Tersimpan
Pengguna memulai dengan mengimpor file simulasi (.csv) atau aplikasi secara otomatis memuat data simulasi terakhir dari penyimpanan browser. Seluruh set data dimuat ke dalam tabel Log Simulasi.
