Explosion Consequence Screening with TNT Equivalent

Halaman ini menyiapkan homepage + UI kalkulator untuk evaluasi konsekuensi & zonasi keselamatan berbasis pendekatan TNT-equivalent: perhitungan massa TNT ekuivalen, scaled distance, kurva overpressure ilustratif, dan ekspor ringkas untuk pelaporan.

Output: WTNT, Z, curve Mode: screening Export: CSV / JSON UI: responsive + SEO
Catatan penggunaan (safety):
Tool ini dimaksudkan untuk studi keselamatan/akademik dan perlu divalidasi dengan literatur/standar yang kamu gunakan. Korelasi overpressure di bawah dibuat ilustratif dan disediakan sebagai hook agar kamu bisa memasang rumus valid (Crowl/Alonso/Sadovski) dari referensimu.

Methods (plug-in ready)

Pilih metode → sambungkan rumusmu di fungsi JS

Energy-based TNT Equivalent

Basis energi: WTNT = η × (m × ΔH) / ΔHTNT. Cocok untuk screening awal & perbandingan skenario.

Crowl / CCPS-style (hook)

Siapkan parameter seperti faktor efisiensi, basis energi, dan korelasi Po vs Z sesuai referensi yang kamu pakai.

Alonso / Sadovski (hook)

Pasang korelasi berbasis scaled distance untuk side-on overpressure / impulse jika diperlukan.
